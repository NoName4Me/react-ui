# Icon

基础图标组件，所有图标源文件来自 [Font Awesome](https://fontawesome.com/)。

## 示例

可以通过 `type`、`size`、`color` 来控制图标类型、大小、颜色。

## 自定义

可以添加的事件如 `onClick` 来当做按钮使用，配合 `extraCls` 控制样式。

```css
/* extraCls 样式 */
.DiyIconStyle {
  color: hsl(350, 80%, 80%);
  transition: all 0.3s ease;
}
.DiyIconStyle:hover {
  color: hsl(350, 90%, 70%);
}
```
