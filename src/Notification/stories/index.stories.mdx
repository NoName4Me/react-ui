# Notification

提供基础的通知组件。

## 通知类型

一共有 `'error', 'warn', 'success', 'info', 'plain', 'hint'` 六种类型的通知组件：

<Preview>
  <Story id="story-notification--base" />
</Preview>

## 使用方式

```js
import { NotificationManager } from 'paper-react-ui';

NotificationManager.warn('Warn title');
NotificationManager.warn({
  title: 'Warn title',
  content: 'you just triggered a danger action',
});
```
